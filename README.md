# 🎰 크립토 트레이더 — 당신은 살아남을 수 있는가?

> **"뉴스를 믿었을 뿐인데, 전 재산을 잃었습니다."**

실제 투자자 1,400만 명 중 **60%가 손실**을 봅니다.
당신도 그 60%에 속할까요?

---

## 💀 이 게임을 해야 하는 이유

### 1. 진짜 돈을 잃기 전에, 여기서 먼저 잃어보세요
$1,000 가상 자금으로 시작합니다.
20턴 뒤, 당신의 통장에 얼마가 남아있을까요?

### 2. 뉴스는 거짓말을 합니다
호재 뉴스가 떴다고 매수했는데 **폭락**합니다.
악재가 터졌는데 오히려 **급등**합니다.
실제 코인 시장과 똑같습니다.

### 3. 당신의 투자 실력을 냉정하게 평가합니다
게임 종료 후, **실제 개인투자자 수익률 분포** 기반으로
당신이 상위 몇 %인지 알려줍니다.
1,400만 투자자 중 당신의 위치는?

### 4. 생각보다 어렵습니다
- 📰 매 턴 3개의 뉴스 — 진짜일까, 가짜일까?
- 💀 갑작스러운 상장폐지 — 보유 코인이 0원이 됩니다
- 📉 인플레이션 — 가만히 있으면 현금이 녹습니다
- 🎭 군중심리 — 모두가 사면 이미 늦었습니다

---

## 🕹️ 플레이 방법

| 단계 | 설명 |
|------|------|
| 1️⃣ | 8개 가상 코인 중 투자할 코인 선택 |
| 2️⃣ | 뉴스를 분석하고 매수/매도 주문 |
| 3️⃣ | 제출 버튼으로 주문 체결 → 다음 턴 |
| 4️⃣ | 20턴 후 최종 자산으로 등급 확인 |

⏱ 1턴 = 60초. 시간 안에 판단하세요.

---

## 🏆 당신의 등급은?

| 등급 | 조건 |
|------|------|
| 🟣 최상류층 | 상위 1% — 전설적 투자자 |
| 🔵 상류층 | 상위 10% — 프로 트레이더급 |
| 🟢 고소득층 | 상위 20% — 탁월한 감각 |
| 🟡 중산층 | 상위 40% — 평균 이상 |
| 🟠 저소득층 | 상위 60% — 분발 필요 |
| 🔴 빈곤/극빈층 | 하위 40% — 뉴스에 속았군요 |
